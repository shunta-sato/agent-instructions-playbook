## Decision log

- 2026-07-05: `.claude/skills` uses **symlinks**, not copies — copies drift (the exact failure of `.github/prompts/`); symlinks cannot.
- 2026-07-05: Catalog `smoke_eval` values are **evidence-backed**: `claude-sonnet-5` = passed (Phase 1-2 recorded runs), `claude-fable-5` = passed (Phase 1-3 supervision), `claude-opus-4-8` = **pending** with a documented smoke procedure — the playbook's own claim-boundary rules apply to the playbook itself. Until the flip, supervisor routes resolve to fable-5 with an explicit `candidate_smoke_eval_not_passed:claude-opus-4-8` fallback reason (visible, not silent). The `playbook-reviewer` agent's first completed run doubles as the Opus smoke eval and its contract says how to flip the catalog.
- 2026-07-05: Lockfile is generated by resolving every task class through the existing `resolve_model_route.py` (no second resolver implementation).
